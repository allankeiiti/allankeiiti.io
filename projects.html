WebScraping sobre o Jogo Eletrônico Streets Of Rage 4

Material de Estudo | Python | 2020
Resolvi aplicar os conhecimentos obtidos sobre WebScraping utilizando o jogo Streets Of Rage 4 como tema principal e com isso saber a forma como usuários opinam referente ao jogo, se são muito técnicos, o que mais citam em suas opiniões. Capturei as opiniões dos usuários do MetaCritic utilizando Selenium para salvar os arquivos HTML e BeautifulSoup para extração final dos dados, a extração das reviews de usuários da Steam foi utilizado a biblioteca steamreviews. Aproveitei para aprender como é documentado um projeto Python utilizando o Sphinx e então, aprender a análise de sentimentos com a biblioteca NLTK. Afirmo que esta brincadeira foi o pontapé para criação de projetos profissionais.

Ferramenta do Suporte N1

Projeto Solo | Java | 2019

Focado em agilizar o atendimento dos chamados do Suporte, o sistema possui a funcionalidade de instalar softwares, fechar programas e tarefas ou executar outros comandos remotamente. A Ferramenta do Suporte N1 foi criada logo após eu finalizar o Script de Compactador de Arquivos citado logo abaixo. A meta era de concluir o sistema em menos de uma semana, e ela foi atingida. Para isso o curso que fiz anos atrás de Java com Banco de Dados do Professor José de Assis foi de grande utilidade, pois me introduziu ao Java Swing que, apesar de já não ser utilizado e sendo substituído pelo JavaFX, a criação de interfaces é bastante simples e útil.

Auto Compactador de arquivos

Projeto Solo | Python | 2019
Durante meu tempo na equipe de suporte da empresa M.L. Gomes, foi solicitado a criação de um script para a equipe de infraestrutura, que dado a quantidade de dias que um arquivo não era acessado, é gravado uma cópia dele em outro diretório e então ele é compactado junto com os outros arquivos que também não foram acessados, seguindo a mesma estrutura de pastas. Terminado a compactação, é gerado um arquivo log da operação, apontando quais foram os arquivos listados, o total de bytes, kbytes, mbytes, gigabytes (Mudando a grandeza conforme a quantidade) e um gráfico de barras com a quantidades de arquivos gravados e compactados por extensão. Foi meu primeiro contato mais a fundo com Python e com a biblioteca MatplotLib. Infelizmente o Script não foi utilizado pela equipe de infraestrutura no final, mas foi essencial para meu aprendizado e sendo meu primeiro projeto criado e finalizado para uma organização.
